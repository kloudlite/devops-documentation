# Image pull secrets

You can create image pull secrets using below formats:

{% tabs %}
{% tab title="Params" %}
Below are the steps to create image pull secrets using params format:

```
# Navigate to image pull secrets
dashboard > settings > image pull secrets

# create image pull secret
create image pull secret > select params format > add required fields
```

{% tab title="Docker config json" %}
Below are the steps to create image pull secrets using docker config json format:

```
# Navigate to image pull secrets
dashboard > settings > image pull secrets

# create image pull secret
create image pull secret > select docker config json format 

# Add required fields
add required fields > Add
```
